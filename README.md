# Clone
```cd ~/go/src/```<br/>
```git clone https://github.com/perthcpe23/nginx-auth-request-ldap.git```

# Build
```cd nginx-auth-request-ldap```<br/>
```go build -o nginx-lda.run nginx-ldap-proxy.go```

# Run
```./nginx-lda.run```
